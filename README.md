<h1>Welcome to BabySprinkle</h1>
<p>BabySprinkle is virtual baby shower destination, where the expectant couple can connect with friends and family.
            When guests RSVP for the virtual baby shower they are entered into the database with their name, email, and username.
            The username guests create is used to identify the guest when they participate in the activities at the virtual baby shower, such as the Guess The Date game.</p>
<h4>Site Details:</h4>
<p>This site was created for COMP 2084 Server-Side Scripting with ASP.NET at Georgian College, Fall 2020.</p>
<p>This site is created using ASP.NET Core, SQL Server, Entity Framework, and is deployed to Azure via GitHub.</p>
<p> View the site at: <a href="https://babysprinkle.azurewebsites.net/" target="_blank">https://babysprinkle.azurewebsites.net/</a> </p>
